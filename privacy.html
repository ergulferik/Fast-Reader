Privacy Policy

Last updated: 2025-11-02

Overview

Fast Reader is a lightweight Chrome extension whose sole purpose is to help users read text faster by displaying selected text word-by-word in a fullscreen, distraction-free view. This extension does not collect, store, or share personal user data.

Data collection

Fast Reader performs all processing locally in the user's browser. The extension does not:

  • Collect or transmit selected text to any remote servers.
  • Store any personal data on external servers.
  • Use cookies or third-party trackers for analytics or advertising.

Permissions and why they are required

The extension requests only the permissions necessary for its single purpose:

  • activeTab — to read the currently selected text on the active tab so it can be displayed in the reader interface.
  • contextMenus — to add the right-click menu entry (e.g. “FRead ile”) that allows users to quickly open the reader for selected text.
  • scripting — to inject a small, minimal content script that collects the selection and creates the reader overlay (all executed locally).
  • storage — to save local preferences such as reading speed or font size on the user's device only.
  • Host permissions (if requested) — limited to the sites where the user enables the extension, required to allow the extension to operate across web pages the user visits.

Remote code

This extension does not download or execute any remote code. All functionality is contained within the packaged extension files.

User controls

Users control the extension at all times: the reader opens only when a user selects text and triggers the extension (via the toolbar, right-click menu, or popup). Users can close the reader immediately and change options such as speed and font size. Preferences saved with chrome.storage remain on the user's device unless the user clears their browser data.
